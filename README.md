# FMCG-Company-Performance-Analysis
Цель проекта - построение информационного дашборда на основе синтетического набора данных о продажах FMCG-компании.
Тема - изучение основных показателей компании в целом и в частности анализ коммерческих показателей по магазинам, категориям и в разрезе поставщиков.
Структура: таблица orderds, stores, suppliers, suppliers_contract_rebates, sku.
Структурно выделяются 4 этапа: 
1. Профайлинг набора данных (оценка полноты данных, анализ распределения метрик) Jupiter Notebook
2. Очистка (преобразование типов данных) POWER BI
3. Создание метрик в POWER BI
4. Моделирование (установление связей) POWER BI
5. Построение дашборда POWER BI
