В папке docs расположены 4 шаблона дашбордов в формате PNG:

1. dashboard-company-performance, dashboard-stores-performance, dashboard-suppliers-performance созданы на основе одного набора данных с тем отличием, что:
   - dashboard-company-performance предназначен для оценки общего состояния ритейлера; основные пользователи - топ-менеджера, директора/руководители департаментов;
   - dashboard-stores-performance предназначен для оценки эффективности отдельных точек продаж и отвечает запросам директоров и менеджеров магазинов;
   - dashboard-suppliers-performance предназначен для планирования работы с поставщиками; основные пользователи - руководители департаментов, категорийные менеджеры.
2. dashboard-hotel-management-admin создан для ежедневного ревью загруженности и дохода отеля;
   dashboard-hotel-management-overview создан для стратегического управления ключевыми показателями.

Файлы .pbix (дашборды созданы в Power BI) и другие данные, связанные с проектами, находятся в отдельных папках.
